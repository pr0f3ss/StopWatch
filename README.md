# RuneLite Stopwatch & Countdown Timer Plugin

A stopwatch and countdown timer plugin for speedrunning, skilling, and PVM encounters in Old School RuneScape.

---

## Features

### Stopwatch
- Start, stop, and reset a stopwatch
- Ideal for speedrunning and tracking PVM encounter durations

### Countdown Timer
- Set a custom countdown
- Your last used timer value is saved and restored automatically across sessions

**Built-in Presets**
- Cerberus
- Inferno 1st/2nd set
- Birdhouse Runs

**Custom Presets**
- Create and name your own timer presets
- Up to 10 custom presets supported
- Click any preset to instantly load it into the timer; delete individual presets at any time

**Sound Notifications**
- Optional sound alert when the countdown reaches zero
- Configurable volume in the plugin settings

---

## Installation

1. Open the RuneLite client and navigate to the **Plugin Hub**
2. Search for **Stopwatch** and click Install
3. The plugin panel will appear in your RuneLite sidebar

---

## Usage

**Stopwatch tab**
- Press **Start** to begin timing, **Stop** to pause, and **Reset** to clear

**Countdown tab**
- Set your desired time using the **min** and **sec** spinners
- Press **Start Countdown** to begin; press **Cancel** to abort and reset to your set time
- Use a **Built-in Preset** or one of your **Custom Presets** to quickly load common times
- Add a custom preset by entering a name and time, then pressing **+**

---

## Configuration

| Option | Description |
|---|---|
| Play sound on completion | Plays an alert sound when the countdown reaches zero |
| Alert volume | Controls the volume of the completion sound |
